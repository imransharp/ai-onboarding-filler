# AI Onboarding Form Filler

Uploads customer PDF — AI extracts all fields for Zong Track portal automatically.

## What it does
- Upload any customer onboarding PDF
- AI reads and extracts all fields
- Auto-detects territory from city name
- Copy each field with one click into portal

## Fields Extracted
Client Name, NTN, POC Name, MSISDN, Email, City, Territory, Employees, Service Type

## Built With
HTML, JavaScript, PDF.js, Groq AI

## Real Use Case
Built for Zong Track customer onboarding — saves 10 minutes per customer
